## 📖 Overview

This project presents an interactive **Sales Performance Dashboard** built using **Tableau** with the Sample Superstore dataset.

The dashboard helps analyze overall sales, profit, orders, category performance, regional sales, and monthly sales trends.

---

## ✨ Dashboard Features

- 📌 Total Sales KPI
- 💰 Total Profit KPI
- 📦 Total Orders KPI
- 📊 Sales by Category
- 🌍 Sales by Region
- 📈 Monthly Sales Trend
- 📉 Profit by Category

---

## 🛠️ Tools Used

- Tableau Public
- Microsoft Excel
- Data Visualization

---

## 📈 Key Insights

- Technology generated the highest sales.
- Technology category earned the highest profit.
- West region recorded the highest sales.
- Monthly sales peaked during November.
- Furniture generated comparatively lower profit.
